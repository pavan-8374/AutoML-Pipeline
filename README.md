# AutoML-Pipeline
A Generic AutoML Pipeline and API
# The architecture of my project
Automated Data Engineering: Handling messy data, fixing headers and dynamically dealing with missing data.

Preprocessing: Automatically separating text from numbers and preparing them mathematically without crashing memory.

AutoML Training: Racing different algorithms against each other and handling imbalanced datasets.

Model Saving: Saving the trained model in a file to the hard drive so it doesn't have to relearn everything from scratch.

Final Goal: Giving non-technical users a way to upload new data, align the features, and get instant predictions.
